Add New Student
Using this feature user can Add new students to the system. In this form all fields are mandatory to fill.
  • Click 'Add New Student' Button
  • Fill the pop-up form
  • Click 'Save Details' to Save
  • Click 'Close' Button to Close
Modify/Delete Student Record
Using this feature user can Modify students details or Delete student record from the system.
  • Click 'Modify/Delete' Button
  • Search Record and and click record to Modify/Delete
  • Click 'Close' Button to close
Add Payment
Using this feature user can Make Payment to the students.
  • Click 'Make Payment' Button
  • Select student and payment category
  • Commit the payment by clicking on 'Pay' button
  • Click 'Close' Button to close
Reports
Using this feature user can view Reports, ex: Fees payment & etc.
  • Click 'View Reports' Button
  • Select fee category & change value
  • Click 'Save' button
  • Click 'Close' Button to close
System Entries
Using this feature user can add new classes & edit/delete, ex: Classes, Events & etc.
  • Click 'Manage Entries' Button
  • Enter new options
  • Click 'Save' button
  • Click 'Close' Button to close
Admin Settings
Using this feature admin can Create new users, Delete users & Reset passwords.
  • Click 'Admin Options' Button
  • Enter new option
  • Click 'Save' button
  • Click 'Close' Button to close
