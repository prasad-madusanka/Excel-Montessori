Add new student
Using this feature user can add new students to the system.
  • Click 'Add New Student' Button
  • Fill the pop-up form
  • Apply Payment categories form the system
  • Click 'Save' Button
Delete/Modify Student Record
Using this feature user can Modify students details or Delete student record from the system.
  • Click 'Modify/Delete' Button
  • Search Record and and click record to Modify/Delete
  • Click 'Close' Button to close
Add payment
Using this feature user can Make Payment to the students.
  • Click 'Make Payment' Button
  • Select student and payment category
  • Commit the payment by clicking on 'Pay' button
  • Click 'Close' Button to close
Change Fees
Using this feature user can change Fees, ex: Admission fess, Monthly Fees & etc.
  • Click 'Change Fess' Button
  • Select fee category & change value
  • Click 'Save' button
  • Click 'Close' Button to close
